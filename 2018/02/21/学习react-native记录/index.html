react native45版本开始后，新建应用会抛出UNpackage xxxx错误，原因是最新的RN无法下载到4个tar.gz文件，由于被墙的原因，所以要手动去下载，下载好后把下载好的四个文件放到路径：/Users/xxx/.rncache/下面，这四个文件的下载路径分别是：
1、folly
2、double-conversion
3、glog
4、boost
